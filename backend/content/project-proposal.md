# Project Proposal: AI Resume Chat Website

## Goal
Create an interactive portfolio that converts passive resume skimming into a grounded, conversational evaluation. Hiring managers can ask questions, explore detailed experience context, and run a bidirectional fit assessment.

## Core Features
- AI chat grounded in resume and project notes
- Expandable experience entries with situation-action-outcome context
- Skills dashboard with strengths, moderate capabilities, and gaps
- Job-fit analysis with evidence and honest gaps

## Design Principles
- Evidence-based responses over generic claims
- Fast streaming responses and clear fallback states
- Professional, dark-themed presentation
